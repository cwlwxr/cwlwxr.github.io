这是由于系统检测到可疑活动，为了防止误用、滥用或垃圾邮件，你的OpenAI 帐户将被阻止。本指南将为你介绍如何解决这个问题。

为什么会出现 Sorry, you have been blocked 错误信息？

你被禁止使用 ChatGPT 或 的原因，可能有以下几种情况：

  1. 系统检测到你的帐户中存在可疑活动；
  2. 你使用了网络代理软件的IP地址，是OpenAI不支持的国家地区。
  3. 你滥用、发送垃圾内容或滥用服务；
  4. 网络防火墙或防病毒软件阻止你访问 ChatGPT。
  5. 你使用其它接码平台的手机号码被其它人滥用注册

如何 Sorry, you have been blocked 错误问题？

如果你遇到 “抱歉，你已被阻止” 的错误信息，可以采取以下几种方法来解决这个问题：

  • 首先，请确认你的账号没有违反 OpenAI 的使用规则，例如未经授权使用 API 或滥发垃圾邮件等行为。
  • 如果你认为账号被错误地封锁，你可以尝试向 OpenAI 官方发送反馈，说明你的情况并请求解封。
  • 你也可以尝试更改你的 IP 地址，因为如果你的 IP 地址被 OpenAI 认为是垃圾邮件发送者的 IP 地址，你的账号也可能会被阻止访问。

重启网络代理服务软件连接

  • 如果你使用了网络代理服务软件，你需要重新连接才能访问 ChatGPT。

清除浏览器的缓存和 Cookie

  • Chrome：点击 Chrome 右上角的三个点，选择“更多工具”，然后选择“清除浏览数据”，清除“Cookie 和其他站点数据/缓存的图像和文件”，最后点击“清除数据” ▼
  • Edge：点击 Edge 右上角的三个点，选择“设置”，然后选择“隐私和服务”，选择要清除的内容，清除“缓存的图像和文件/Cookie 和其他站点数据”，最后点击“清除”。
  • Firefox：点击 Firefox 菜单，选择“设置”，然后选择“隐私和安全”，选择“Cookie 和站点数据”，最后点击“清除”。

使用不同的浏览器

  • 尝试使用不同的浏览器，例如 Chrome、Microsoft Edge、Firefox 或 Brave 等……来访问 ChatGPT。
  • 如果你在桌面上使用 ChatGPT，请尝试在 Safari 或 Chrome 的移动设备上使用它。

禁用你的 Chrome 谷歌浏览器扩展程序

  • 在 Chrome 上，单击拼图图标，选择 “管理扩展”。
  • 删除或禁用任何不需要的、可疑的和网络代理服务扩展。
