Telegram机器人自动推送推特YouTube同步消息到频道群组

2026-05-29

本文链接：https://www.chenweiliang.com/cwl-1925.html

Telegram频道理论上是无限受众的广播消息的重要工具。

至于Telegram群组（支持多达 200,000人 的社区并且还在不断增加），它们提供类似的功能，可以让加入的成员直接相互交流等。

  • Telegram频道和群组本身可能需要大量工作来管理，除非你不打算长期保持关注者，否则你需要定期分享新鲜内容。
  • 有些Telegram频道选择在创建定期原创内容的基础上，转发其他频道和群组的内容。可以想象，这是爱的劳动。
  • 更加明智之举是创建你自己的 Telegram 机器人来自动化该过程。

创建 Telegram 机器人不需要机器人技术或类似学位。任何电脑小白菜鸟都可以在 10 分钟左右的时间内启动并运行机器人。你甚至不需要任何编码经验，我不骗你。

在这篇文章中，我们将向你展示如何创建一个 Telegram 机器人来自动发布来自 Twitter、YouTube、VK 和 RSS 源的感兴趣的文章，并与你的大量订阅者/成员一起发布。从各方面来看，这胜过必须手动共享此内容。

第 1 步：创建Telegram频道

  1. 只需转到 Telegram 应用程序左上角的菜单即可。这适用于智能手机和桌面应用程序。
  2. 选择“新频道”（带有广播图标的频道）。
  3. 继续并输入你的频道名称和可选的相应频道描述。
  4. 根据你的意图，你可以将其设置为公共或私有。作为公共频道，用户可以使用他们的搜索框找到它。另一方面，私人频道需要邀请链接才能加入。

第 2 步：为你的 Telegram 频道/群组创建一个 Telegram Bot机器人


就如 Telegram 官方说的那样，BotFather是一个可以统治所有人的机器人。这是创建新机器人和管理现有机器人时的起点。那么，这就是我们的下一站。

  1. 打开BotFather。在 Telegram 搜索框中输入 Botfather。单击它以打开机器人。
  2. 输入/newbot命令以创建新机器人。为你的新机器人选择一个名称。除非你打算将其作为公共机器人，否则名称并不重要。我们的机器人将在幕后运行节目。
  3. 现在为你的新机器人选择一个用户名。用户名的长度可以介于 5 到 32 个不区分大小写的字符之间。通常，用户名必须以后缀 -bot 结尾，例如： etufo_bot。
  4. 完成后，你将收到一个 HTTP API 令牌。即类似于：
    435074775:AAHRQTtAOhQ1POBw9L98ru6Giek0qafTvME

    。确保此令牌安全并安全存储。如果有人持有此令牌，他们就可以完全控制你的机器人。

第 3 步：使用 Manybot 机器人自动发布到你的 Telegram 频道/群组

现在我们有了一个功能强大的机器人，我们将使用另一个机器人来简化流程。@Chatfuel_bot 是一个受欢迎的选择，但我个人最喜欢的是 @Manybot。Manybot 将成为你的频道和你创建的机器人之间的链接。要自动发布来自 RSS 源、Twitter 和 YouTube 的内容，请按照以下步骤操作：

  1. 打开 Manybot机器人。
  2. 使用/addbot 命令创建你的第一个机器人。（我们已经这样做了，所以耶！）
  3. 跳过使用 Botfather 创建新机器人的步骤，因为我们已经完成了。
  4. 单击“I’ve copied the API token.。（我已复制 API 令牌）” 在 Botfather 中创建机器人后，复制并粘贴你收到的令牌。
  5. 接受令牌后，写下关于你的机器人的简短描述或跳过此步骤。
  6. 你的机器人现已准备就绪！选择“向订阅者发送新帖子”。

你将被发送到新创建的机器人。从这里你可以向订阅者发送新帖子、部署自定义命令和表单回复等等……但现在让我们保持简单。转到底部的设置。你将看到以下内容：频道/自动发布/时区/取消。

  1. 单击“频道”开始。
  2. 选择“添加频道”
  3. 输入频道名称/链接。例如外星人UFO真相或 https://t.me/etufoorg

哎呀！我们将在这一点上遇到障碍。

  1. 所以，让我们回到我们的频道。
  2. 我们将我们的机器人设置为管理员。
  3. 为此，我们导航到频道设置，然后导航到管理员。
  4. 然后我们将我们的机器人添加为管理员。

现在继续……

  1. 返回你的机器人并添加你的频道。
  2. 完成后，选择“返回”
  3. 选择“ Autoposting ”
  4. 选择内容来源，即 Twitter(@username)、YouTube 频道、VK 和 RSS 提要（例如提要: https://www.etufo.org/feed ）
  • 成功！
  • 温馨提示：Telegram机器人自动同步到频道或群组不是即时的，需要一段时间（大概1~2个小时）抓取。
