2024-10-27

本文链接：https://www.chenweiliang.com/cwl-30363.html

OpenAI对亚洲地区的用户进行了大规模封号，尤其是针对台湾、日本和香港的IP地址，命中率高达40%，新注册的ChatGTP账号和Plus也不能使用。

随着封禁的数量越来越多，人们发现封禁似乎没有针对性，无论是购买Plus 还是仅使用API，都可能被封禁。

OpenAI官方尚未做出说明，因此封禁原因不确定，主要分为“大规模注册”和“API调用被封”两种类型：

  1. 一方面，很多人都认为封禁的重灾区是之前的大规模注册账户。不少网友表示他们手动注册的账号还在，但一些在电商平台购买的账号已经被停用。
  2. 另一方面，有人猜测之前的账户行为可能存在API滥用嫌疑，例如一个人使用多个账户获取API服务，如果检测到在同一IP或类似IP上不断更换API并发送请求，就会违反OpenAI API使用规则。

2023年4月1日是OpenAI API Key免费调用额度到期日，有人担心这几天会有大量账户购买令牌，导致API被滥用，因此先封禁了一批账户。

除了这些与账户本身有关的原因外，还有人从ChatGPT的用处切入分析，认为ChatGPT可以提高工作效率，因此账户被封禁后工作效率大幅下降。

如何判断OpenAI账户是否被封禁？

需要注意的是，许多账户仍可以如常使用，因此不要过度恐慌，要仔细分辨自己的账户是否还能使用。

判断方法大致如下：

如果登录状态被封禁，出现“无法加载历史信息”的提示，或“输入框输入的内容无法发送”的情况，可能会封禁账户。

如果不是在登录状态下被封禁，那么在登录过程中就会报错：

“账户停用。如果你需要帮助，请通过我们的帮助中心与我们联系。”（错误=账户停用）。

如果出现“无法加载历史信息”或“无法发送输入框中的内容”的提示，则可能是账户被封的原因。

但若出现其它错误提示，例如“OpenAI's services not available in your country”等……则不一定是账户的问题。

ChatGTP账号为什么被封？

有人推测 API 调用被封的原因是之前账户的行为存在 API 滥用嫌疑。

例如，如果一个人通过多个账户获取 API 服务，并被检测到在同一IP地址或类似IP地址下，不断更换API发出请求，就会违反OpenAI API使用规则。

  • 此外，2023年4月1日是OpenAI API Key免费调用额度到期的日子。这种猜测并非毫无根据，因为之前Midjourney已经停止了免费使用，部分原因是因为过度使用。
  • 2023年3月28日，Midjourney的创始人兼CEO表示，许多人为了避免付费，注册大量新账户只使用免费额度，这加剧了GPU短缺的情况，导致付费用户的服务受到影响。
  • 当然，这些都是基于账户本身存在问题的猜测。

据悉，由于需求量过大，ChatGPT甚至还出现停止Plus付费的情况。

虽然ChatGPT已经越来越受欢迎，但由于它是由美国公司开发的，我们使用它时很容易受到限制。

每个月底，OpenAI都会有大动作，他们进行了一次风控调整，封禁了一批账户，他们对很多亚洲地区的账户进行了封禁，甚至有一些Plus账户也被封禁了。

问：若ChatGPT账号被封，原因有哪些？

答：ChatGPT是一个由OpenAI开发的AI语言模型，它不是一个实际的账号，因此无法被封禁。但是，如果使用ChatGPT的API或服务的账号被封禁，可能的原因有：

  • 违反服务条款：如果用户在使用ChatGPT的API或服务时违反了OpenAI的服务条款，OpenAI可能会将其账号封禁。
  • 滥用：如果用户滥用ChatGPT的API或服务，例如频繁请求或使用ChatGPT进行不良行为，OpenAI可能会将其账号封禁。
  • 付款问题：如果用户使用付款问题或诈骗方式使用ChatGPT的API或服务，OpenAI可能会将其账号封禁。

总之，OpenAI会严格审核用户使用ChatGPT的API或服务的行为，以确保它们不会被滥用或用于不当用途。
