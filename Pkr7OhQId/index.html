如果你在使用 ChatGPT 的过程中收到“You must sign up for an OpenAI account before continuing”的错误信息，那么你需要采取一些措施来解决这个问题。在这篇文章中，我们将会告诉你如何解决这个问题。

ChatGPT 中的“error=signup required”是什么意思？

在 ChatGPT 中，当你看到“error=signup required”时，这意味着你没有登录到 OpenAI账号。

You need an OpenAl account

You must have an OpenAl account to use ChatGPT. If you don't

have one, sign up now and return to chat.openai.com to log in

and continue.


Cancel

Create an OpenAl account

要使用 ChatGPT，你需要一个 OpenAI 帐户，如果你没有注册，则无法使用 ChatGPT。

如果你已经登录，但是仍然遇到了错误，那么可能是由于网站出现了错误或故障。

如何解决“You must sign up for an OpenAI account before continuing”？

要解决“You must sign up for an OpenAI account before continuing”的错误信息，你需要注册一个 OpenAI 帐户。

下面是具体步骤：

  1. 前往 https://chat.openai.com/auth/login 
  2. 单击“注册”按钮。
  3. 输入你的电子邮件地址和手机号码并单击“继续”。
  4. 创建一个密码并单击“继续”。
  5. 确认你的邮件地址。
  6. 输入你的名字和姓氏并单击“继续”。
  7. 验证你的手机号码，并发送短信验证码。

    ChatGPT哪些国家能用？OpenAI账号可以支持哪个地区？

    • 美国、加拿大、英国以及亚洲和欧洲的大部分地区都可以使用 ChatGPT。
    • 请注意，ChatGPT 在某些国家/地区不可用，例如阿塞拜疆、沙特阿拉伯、香港等地……
    • 如果你位于这些地区，你需要使用来自其它具有 ChatGPT 访问权限的国家/地区的手机号码来注册ChatGPT 帐户。

      清除浏览器的缓存和 Cookie

      如果你已经登录ChatGPT，但仍然收到错误消息，则需要在使用 ChatGPT 之前清除浏览器的缓存。

      • Chrome：点击 Chrome 右上角的三个点，选择“更多工具”，然后选择“清除浏览数据”，清除“Cookie 和其他站点数据/缓存的图像和文件”，最后点击“清除数据” ▼
      • Edge：点击 Edge 右上角的三个点，选择“设置”，然后选择“隐私和服务”，选择要清除的内容，清除“缓存的图像和文件/Cookie 和其他站点数据”，最后点击“清除”。
      • Firefox：点击 Firefox 菜单，选择“设置”，然后选择“隐私和安全”，选择“Cookie 和站点数据”，最后点击“清除”。
