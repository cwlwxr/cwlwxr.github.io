ChatGPT 中的“Err_too_many_redirects”是什么意思？

This page isn't working
xxxxx redirected you too many times.
Try clearing your cookies.
ERR_TOO MANY REDIRECTS
redirected you too many times.
Reload

ChatGPT中的“Err_too_many_redirects”指的是重定向过多的错误。

当服务器过载时，可能会出现“Err_too_many_redirects”错误。

此外，互联网连接太慢或不活动也可能导致出现此错误。

如果ChatGPT使用者太多，也可能会出现性能问题，导致无法给予答复。

你可以通过访问 https://status.openai.com/ 来监控ChatGPT的状态。

如果绿色条显示，表示网站如常运行，而深色条则表示出现中断。

为避免出现“Err_too_many_redirects”错误，你可以尝试减少访问ChatGPT的次数或等待一段时间再尝试。

如何解决ChatGPT提示err_too-many_redirects错误问题？

在使用 Chat GPT 时出现了“err_too-many_redirects（此页面无法如常工作，请尝试清除你的 cookie）”的错误提示，那么你需要采取一些措施来修复它。以下是几种解决方法：

解决方法1：重新启动网络代理软件

  • 有时，网络代理程序可能会导致 ChatGPT 显示“err_too-many_redirects”错误。
  • 如果你连接了网络代理，但仍然遇到err_too-many_redirects 错误，请断开重启，然后尝试重新登录 ChatGPT。

解决方法2：检查 OpenAI 的状态和等待

在使用 Chat GPT 之前，你可以先前往 https://status.openai.com/ 检查 OpenAI 的状态 ▼

  • 如果绿色条显示“网站完全如常运行”，那么这个错误可能是由于服务器过载而引起的。
  • 此时，你需要等待一段时间，直到服务恢复如常。

解决方法3：清除浏览器的缓存和 Cookie

  • Chrome：点击 Chrome 右上角的三个点，选择“更多工具”，然后选择“清除浏览数据”，清除“Cookie 和其他站点数据/缓存的图像和文件”，最后点击“清除数据” ▼
  • Edge：点击 Edge 右上角的三个点，选择“设置”，然后选择“隐私和服务”，选择要清除的内容，清除“缓存的图像和文件/Cookie 和其他站点数据”，最后点击“清除”。
  • Firefox：点击 Firefox 菜单，选择“设置”，然后选择“隐私和安全”，选择“Cookie 和站点数据”，最后点击“清除”。

Firefox：依次点击“Firebox > 菜单 > 设置 > 隐私和安全 > Cookie 和站点数据 > 清除”。

解决方法4：使用不同的浏览器

  • 尝试使用不同的浏览器，例如 Chrome、Microsoft Edge、Firefox 或 Brave 等，来访问 Chat GPT。
  • 如果你在桌面上使用 Chat GPT，请尝试在 Safari 或 Chrome 的移动设备上使用它。

解决方法5：注销并重新登录到 Chat GPT

  • 如果以上方法都无法解决问题，那么你可以尝试退出 Chat GPT 并重新登录。
  • 单击左侧栏中的“退出”，然后重新登录 Chat GPT 并再次尝试使用它。

解决方法6：注册一个新帐户

如果你受到速率限制，你可以注销当前的 ChatGPT 帐户并单击“注册”按钮，使用另一个电子邮件地址/手机号码注册一个新帐户，具体方法可参考以下教程 ▼
