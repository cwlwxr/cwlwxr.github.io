到底eSIM卡是什么？

首先，插在手机里面的 SIM 卡，应该是大家都很熟悉的东西。

但它除了在购买新手机或是申请新手机号码时，才会被你看到之外，其他时间几乎都躲藏在你的手机里面。

经过几年来的演进，从传统的大卡(mini SIM)变成中卡(micro SIM)，然后发展到细卡(nano SIM)，这三种都是「实体卡」▼

  • 至于eSIM，全写为Embedded-SIM，中文可称为「嵌入式SIM卡」。
  • 但如果叫「内置SIM卡」可能大家会更好理解。
  • 因为eSIM是可由远端启动的SIM卡规格，没有实体卡槽。

哪些国家地区能用eSIM卡？

目前有支援eSIM功能的智慧型手机，分别是Google Pixel 2系列，以及苹果旗下的iPhone XS( 不包括中国内地版行货)、iPhone XS Max( 不包括中港澳版行货)。

  • iPhone XS最多可保存八张eSIM，但同一时间只能使用 1 张 eSIM 。
  • 相信未来也会有越来越多的手机加入 ^_^

在中国大陆可以用eSIM卡吗？

在2018年中国联通获得了eSIM一号双终端业务试点支持，但目前支持eSIM卡服务的中国城市非常少……

因为eSIM卡的出现，用户非常容易更换到其它手机运营商，可能中国手机运营商担心用户流失，所以在中国国内地 Mainland China的iPhone XS、iPhone XS max用户，都不能使用eSIM卡。

激活使用eSIM流程

以下为激活使用eSIM流程简介(How to activate eSIM?)：

  1. 手机必须有支援eSIM功能；The handset must be equipped with eSIM function ；
  2. 电信公司须支援eSIM服务；The Mobile Operator must be authorized to support eSIM ；
  3. 要获得电信公司的专属QR Code来下载安装；Thehandset must scan the QR code issued by the said Mobile Operator ；
  4. 手机用户要输入Confirmation Code密码(可选择Optional) ； The customer may need to input Confirmation Code.

如此一来，以后不仅不需要在手机插上SIM卡，甚至在出国时可以下载当地的eSIM 来上网(当然也要付费)，省却普通卡的配送快递物流费。

之后不使用时只要从手机设定中删除QR Code 即可。
