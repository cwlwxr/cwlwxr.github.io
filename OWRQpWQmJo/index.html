你是否在使用ChatGPT Plus时遭遇到“Authorization error accessing plugins”的烦恼？

这个错误仅会在你拥有配备活跃插件的ChatGPT Plus时才会呈现于你。

一旦你的一个或多个插件遭到撤销授权或未通过验证，此问题将产生。

由此，当你向ChatGPT发送信息时，将无法获得任何回应。

在此指南中，你将理解如何修复ChatGPT Plus上的“访问插件授权错误”，以及其背后的涵义。

为何在使用ChatGPT Plus时访问插件会引发授权错误？

在试图访问ChatGPT Plus上的插件时，你可能会遭遇授权错误，因为OpenAI最近撤销了部分插件的授权，并未经过验证

这需要插件的创作者重新进行验证，以要求用户重新为插件授权。

此错误也可能源自订阅级别不符，或是插件系统内部出现的技术故障。

如果你碰到这一问题，我们建议你找出有问题的插件，并对其进行重新安装。

如何修复ChatGPT上的“Authorization error accessing plugins”？

若要修复 ChatGPT上的“访问插件授权错误”，你需卸载并重新安装未能如常运行的插件（如ScholarAI）。

此后，根据需要重新为这些插件进行授权，并对其进行测试，以验证它们是否能够再度运作。

操作步骤如下：

  1. 登录ChatGPT，并前往插件商店。
  2. 寻找无法如常工作的插件（如Prompt Perfect、ScholarAI）。
  3. 卸载这些无法如常工作的插件。
  4. 重新安装这些插件，并视情况重新进行授权。
  5. 测试这些插件，以验证它们是否重新开始运作。

倘若你拥有多个插件，你可能需逐一进行卸载并重新安装，以找出罪魁祸首。

若你仍屡遇授权错误，你需与插件开发者取得联系，以获得进一步的协助。
