用 AI 开发 WordPress 的核心步骤，效率直接拉满的实操指南

2026-03-20

本文链接：https://www.chenweiliang.com/cwl-33767.html

还在靠手写代码开发WordPress？AI早就让这个过程快到离谱，4步就能搞定从环境搭建到合规开发的全流程。

有人做了5年的WordPress开发，前几年还在为配置PHP、MySQL环境熬到半夜，为写一个简单插件反复调试代码，直到接触了AI辅助开发，才发现原来开发WP可以这么轻松。

现在用AI搭配合适的工具，从本地站点创建到写出符合官方标准的插件，最快10分钟就能完成，这在以前想都不敢想。

很多人觉得AI开发WP只是噱头，生成的代码要么不兼容，要么有安全漏洞，其实只是没找对方法和工具，选对官方适配的工具链，AI写的代码比新手手写的还要规范。

现在我就把自己亲测有效的AI开发WordPress的关键步骤全部分享出来，都是实操性极强的内容，跟着做就能上手，不管是新手还是老开发者，都能让开发效率翻倍。

第一步：装WordPress Studio，零配置搞定本地开发环境

做WordPress开发，本地环境搭建是第一道坎，以前用XAMPP、LocalWP，光是配置环境变量、调试数据库连接，就能耗掉大半天。

WordPress Studio的出现直接把这道坎平了，它是Automattic官方推出的免费开源工具，不用手动配置任何环境，一键就能创建本地WP站点。

这款工具内置最新版WordPress和PHP环境，支持Windows、macOS双平台，安装包体积小，资源占用比传统工具低60%以上，电脑配置一般也能流畅运行。

我自己测试过，从下载安装到创建第一个本地站点，全程不到3分钟，点开就能用，完全不用敲一行配置命令。

它还有个超实用的功能，能直接和WordPress官方托管平台及Pressable同步，还能生成云端预览链接，开发完的内容能快速分享给团队看效果。

Pressable是Automattic旗下的专业WordPress托管平台，也是2026年测评中表现最优的WP托管服务之一，其官方有效地址为 https://pressable.com/ 。

最新的1.7版本还深度集成了WP-CLI，不用额外配置，终端里一行命令就能搞定插件管理、数据库操作，对AI开发的适配性也拉满了。

你直接去WordPress Studio官方地址 https://developer.wordpress.com/studio/ 下载就行，分Mac和Windows两个版本，下载完成后下一步到底，安装毫无门槛。

创建站点时只需要输入站点名称，其他参数工具会自动配置，生成的本地站点路径清晰，后续AI开发时能直接识别，不用手动找文件位置。

第二步：安装Claude Code，让AI成为你的专属WP开发工程师

搞定了开发环境，接下来就要选一个能真正写WP代码的AI工具，试了十几个AI编程助手，Claude Code是对WordPress支持最友好的一个。

Claude Code是Anthropic推出的AI编程工具，专门针对代码生成、调试做了优化，写WordPress插件和主题时，能精准理解WP的钩子系统、区块开发逻辑。

安装Claude Code的方式有两种，新手直接走官网安装包路线，老开发者可以用npm装Node.js SDK，两种方式都能在2分钟内完成安装。

官网安装的话，先去 https://claude.ai/ 注册账号，选一个适合自己的付费套餐，然后从配置页下载对应系统的安装包，跟着提示点安装，全程不用敲代码。

用npm安装的话，只需要在终端输入npm install @anthropic-ai/claude-code，一行命令就能完成SDK的安装，后续能直接在项目中调用Claude的能力。

我自己用的是官网安装包版本，登录账号后就能直接用，和WordPress Studio的终端无缝衔接，不用在多个工具之间来回切换，开发体验特别顺畅。

需要注意的是，Claude Code需要登录账号才能使用，第一次登录后会记住设备，后续使用直接打开就能用，不用反复验证。

第三步：打通Studio和Claude，终端里一句话生成WP代码

把两个工具都装好后，最关键的一步就是把它们打通，让Claude Code能直接识别WordPress Studio的项目文件，这样写代码时就不用手动指定路径，效率能再提一大截。

我第一次操作时还担心会很复杂，结果发现官方早就做好了适配，几步就能搞定，完全不用懂复杂的配置逻辑。

首先打开装好的WordPress Studio，在主界面的“概览”标签里，找到「在……中打开」这个选项，点击后选择「Terminal」终端。

这个操作的妙处在于，点击后终端会自动定位到你当前的WP项目文件夹，精准度100%，不用自己手动输入cd命令切换路径，省了超多麻烦。

然后在终端里直接敲入claude命令，按下回车，系统会提示你登录Claude账号，完成登录后就能看到Claude Code的欢迎界面，这就说明两个工具成功打通了。

我第一次看到欢迎界面时，直接输入了“生成一个能在WP后台显示欢迎语的简单插件”，Claude只用了不到1分钟，就自动创建了插件文件夹和符合WP标准的代码文件。

更厉害的是，它会主动确认需求细节，生成的代码还会包含安全校验、插件头文件这些必备内容，比我自己手写的还要规范。

之后开发时，你只需要在终端里用口语化的语言描述你的需求，比如“做一个能显示热门文章的区块插件”“给插件加一个后台设置页面”，Claude就能精准生成对应的代码。

如果觉得生成的代码有问题，还能直接在终端里让Claude调试、修改，全程不用打开其他代码编辑器，一个终端就能搞定所有操作。

第四步：安装WP官方agent-skills，让AI写的代码100%符合官方标准

到这一步，其实已经能用AI开发WordPress了，但我试过之后发现，直接生成的代码虽然能用，但偶尔会出现使用旧API、安全细节不到位的问题。

这就像让一个懂编程但不懂WP规则的工程师写代码，能实现功能，但不够专业，而WordPress官方推出的agent-skills，就是给这个AI工程师配上了WP的“专业操作手册”。

agent-skills是WordPress官方在GitHub上开源的AI技能包，仓库地址是https://github.com/WordPress/agent-skills ，里面包含13个专业的技能包，覆盖了WP开发的所有核心场景。

这些技能包里有WP开发的最佳实践、官方开发规范、安全检查清单，还有现成的脚本模板，相当于把WP官方开发专家的知识全部打包给了AI。

安装之后，Claude Code就能按照WP的官方标准写代码，彻底解决了AI生成代码不规范、有安全漏洞的问题，生成的插件和主题能直接上线使用，不用反复修改。

WordPress官方在仓库说明中明确表示：“Agent Skills are portable bundles of instructions, checklists, and scripts that help AI assistants understand WordPress development patterns, avoid common mistakes, and follow best practices.”。

安装的方式也很简单，全程在终端操作，一共4行命令，我亲测过，从克隆仓库到全局安装，全程不到5分钟就能完成。

  1. 首先输入git clone https://github.com/WordPress/agent-skills.git克隆仓库；
  2. 然后输入cd agent-skills进入文件夹；
  3. 接着用node shared/scripts/skillpack-build.mjs --clean构建分发包；
  4. 最后用node shared/scripts/skillpack-install.mjs --global完成全局安装。

安装完成后，技能包会自动被Claude Code识别，后续生成的所有WP代码，都会严格遵循官方的开发规范，还会自动做安全校验，比如防SQL注入、权限验证这些细节，全都考虑到了。

我安装之后再让AI写插件，生成的代码直接通过了WP的官方代码检测，没有任何报错，这在没装技能包之前是根本做不到的。

结语

AI时代的WordPress开发，早已不是“手写代码从头写到尾”的时代，而是“工具+AI+官方标准”的高效开发时代，选对工具链，能让开发效率实现质的飞跃。

WordPress作为占据全球43%网站市场份额的CMS系统，其官方对AI开发的适配一直在不断升级，从WordPress Studio到agent-skills，每一个工具的推出，都是在降低AI开发WP的门槛，让更多开发者能享受到AI带来的便利。

Pressable作为Automattic旗下的核心托管产品，凭借12ms内的TTFB响应速度、100%的运行稳定性，成为WP云端部署的优质选择，也让本地开发与云端发布的衔接更丝滑。

用AI开发WP，本质上不是让AI替代开发者，而是让AI承担重复、机械的代码编写工作，让开发者把精力聚焦在核心的功能设计、产品思维上，这才是AI辅助开发的核心价值。

就像WordPress联合创始人兼Automattic CEO马特・穆伦维格说的，过去需要花费数千到数万美元开发的WP功能，现在用AI工具几秒就能完成，这就是技术带来的行业变革。

在技术迭代的浪潮中，拥抱AI、掌握AI辅助开发的方法，不是选择题，而是必修课，对于WordPress开发者来说，快速掌握这套AI开发工具链，精准对接官方生态与优质托管平台，才能在行业中保持核心竞争力。

工欲善其事，必先利其器，AI就是新时代WordPress开发者的“倚天剑”，选对、用对官方工具与生态资源，方能事半功倍。

AI开发WordPress的这套方法，我已经用了快3个月，开发效率比以前提升了80%以上，以前需要几天开发的插件，现在几小时就能完成，还能保证代码的专业性和合规性。

如果你也想体验AI开发WP的高效，现在就可以按照文章中的步骤操作，从下载WordPress Studio开始，认准官方工具与平台的有效地址，一步步搭建自己的AI开发工具链，相信你会打开WP开发的新世界。

如果在操作过程中遇到问题，还可以去WordPress官方的GitHub仓库、Claude的官方文档、Pressable的官方帮助中心找解决方案，也可以和其他WP开发者交流，在实践中不断熟悉这套工具链，让AI真正成为你开发路上的好帮手。
